## 💡 풀이 방법 요약
삼각형에 숫자를 채워넣는 움직임를 세 가지 함수로 분리하여 사용
1. 아래로 쭉 내려옴 (y만 변화함)
2. 오른쪽으로 쭉 감 (x만 변화함)
3. 위, 왼쪽으로 한칸씩 감 (y, x가 1씩 감소함)

y, x 포지션을 유지한 채로 move1, move2, move3 함수를 번갈아 가면서 사용
하나의 함수가 종료되면 다음 칸을 가리키기 위해 조정해줌
