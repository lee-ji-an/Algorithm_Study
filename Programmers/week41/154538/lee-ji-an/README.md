## 💡 풀이 방법 요약
> dp

1. dp 배열을 x ~ y까지 차례대로 탐색 -> dp 배열을 채움
2. `dp[x + n]`에 `dp[x] + 1` 를 저장 / `dp[2 * x]`에 `dp[x] + 1` 저장 / 3 * x 도 마찬가지..

✅ +n / *2 / *3 모두 현재 x 값보다 값을 더 증가시키는 연산이기 때문에  
x부터 증가하는 방향으로 dp를 갱신하면 x 보다 작은 수의 dp값이 갱신될 일이 없음
-> dp 배열을 순서대로 한 번씩만 탐색하면 됨
