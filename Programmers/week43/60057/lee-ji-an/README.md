## 💡 풀이 방법 요약
- 문자열 길이를 1 ~ N 까지 모두 탐색 
1. 이전 문자열과 현재 문자열이 일치할 때
   - 중복 횟수를 증가시킴
   - 처음 중복 문자열이 등장한 경우, 혹은 중복 문자열 등장 횟수의 자릿수가 증가한 경우에만 문자열 길이 +1
2. 이전 문자열과 현재 문자열이 일치하지 않을 때
    - 문자열 길이만큼 총 문자열 길이를 증가시킴
    - 맨 끝 문자열을 탐색할 때에는 문자열 길이가 부족할 수 있기 때문에 `min(l, N - i)`
