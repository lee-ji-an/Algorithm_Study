## 💡 풀이 방법 요약
1. 두 문자열로 쪼개기
   - `(` -> cnt 1 증가
   - `)` -> cnt 1 감소
   - 처음으로 cnt가 0이 되는 지점을 기준으로 분할
2. u가 `올바른 괄호 문자열`인지 판단
   - `(` -> cnt 1 증가
   - `)` -> cnt 1 감소
   - 전체 괄호 문자열을 탐색할 때 cnt가 음수가 되는 경우가 발생하면 `올바른 괄호 문자열`이 아님
3. 재귀를 이용해 `올바른 괄호 문자열`로 변환
   - 문제에 제시된 방법과 동일
