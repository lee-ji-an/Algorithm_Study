## 💡 풀이 방법 요약
> Queue
1. 조건에 맞을 때까지 트럭을 올림
    - 트럭은 deque로 관리 / (다리에 올라간 시간, 트럭 무게) 를 append
2. 트럭 하나 빼기
    - 트럭이 다리를 모두 건너는 시간은 `(다리에 올라간 시간 + 다리 길이)`
    - 모두 건너는 시간이 현재 시간보다 길면 시간 갱신
3. 다시 조건에 맞을 때까지 트럭 올리기
