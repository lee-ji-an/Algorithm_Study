## 💡 풀이 방법 요약
dp를 이용하여 풀었음  

이미 장난감을 기본 부품으로 만들 수 있다면, 추가함  
만약 장난감을 중간 부품으로 만들 수 있다면, 이 중간 부품을 기본 부품으로 만들 수 있는지 확인함.  
만약 중간 부품을 기본 부품으로 만들 수 있다면, 장난감에 기본 부품들을 추가함  
만약 중간 부품을 기본 부품으로 만들 수 없다면, 재귀를 통해 중간 부품을 기본 부품으로 만듦.
