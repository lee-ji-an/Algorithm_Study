1. root에서 길이가 가장 먼 노드를 구함 (dfs 사용, root 아니고 임의의 점이어도 됨)
2. 'root에서 길이가 가장 먼 노드' 에서 길이가 가장 먼 노드까지의 거리 (dfs 사용) -> 트리의 지름
