## 💡 풀이 방법 요약

> 연산자 순서를 정하고 숫자 넣어서 계산하고 최댓값, 최솟값 구하기

1. 연산자 개수대로 추가해서 연산자 문자열 만들기
   * 0 : 더하기 / 1 :  빼기 / 2 : 곱하기 / 3 : 나누기
2. 순열로 연산자 순서 정하기
3. 순서 정할 때마다 숫자 넣고 식 계산해서 최댓값과 최솟값 구하기
