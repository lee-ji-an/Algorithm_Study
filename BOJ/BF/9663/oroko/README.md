## 💡 풀이 방법 요약
> 각 열, 두 대각선에 퀸이 있는지 마킹하고 없으면 놓기

* 각 행에는 퀸 하나만 놓을 수 있음
* 열 : (i, j)에 퀸을 놓으면 j가 같은 곳에 다른 퀸을 놓을 수 없음
* 대각선1 : (i, j)에 퀸을 놓으면 `i+j`가 같은 곳에 다른 퀸을 놓을 수 없음
* 대각선2 : (i, j)에 퀸을 놓으면 `N-(j-i)-1`가 같은 곳에 다른 퀸을 놓을 수 없음

### queen(r)
* r행에 퀸 놓기
* r이 N이면 N개 다 놓았으므로 cnt++
* 0~(N-1)열 중 놓을 수 있으면 놓았다고 표시 후 놓고 표시 제거

### check(i, j)
* (i, j)에 퀸을 놓을 수 있는지 확인
* j열이 true이면 놓을 수 없음
* diag1[i+j]가 true이면 놓을 수 없음
* diag2[N-(j-i)-1]가 true이면 놓을 수 없음
* O(1)로 확인 가능
