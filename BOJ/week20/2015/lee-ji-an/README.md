## Info
[2015 수들의 합](https://www.acmicpc.net/problem/2015)

## 💡 풀이 방법 요약
> 누적합 이용

1. sum_value 에 누적합을 저장
2. i 위치를 끝지점으로 하는 정답이 있는지 확인하고 (`sum_value - K` 가 존재하는지 확인) 정답 횟수에 추가
3. i 위치까지의 누적합을 sum_dict에 추가
4. 0번째 인덱스부터 시작하는 경우의 수를 정답 횟수에 추가

```Python
if K in sum_dict:
    ans += len(sum_dict[K])
```
