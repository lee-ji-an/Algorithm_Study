# 풀이 방법 요약
1. bfs로 root노드에서부터 모든 정점까지의 depth(거리)를 구함
2. input으로 들어온 값을 차례대로 탐색
3. 다음 정점에서 depth가 증가할 때 depth가 깊은 쪽의 노드가 depth가 얕은 쪽 노드의 자식이어야 함
