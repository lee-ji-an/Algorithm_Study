## 💡 풀이 방법 요약

> bfs로 0~K개 부순 경우를 모두 넣고 탐색하기

### bfs
1. 큐애는 좌표, 부순 개수, 거리 저장
2. visited : visited[i][j][k]는 k개 부수고 (i,j) 칸에 방문한 적 있는지 여부
3. 도착한 경우, 벽 부쉈으면 큐에서 꺼낸 cnt 반환
4. 다음 칸 중에서
   1. 벽이 아니거나
   2. 벽인데 K개보다 작게 부쉈으면 부수고
   
    해당하는 칸에 방문 표시해주고 큐에 넣기
