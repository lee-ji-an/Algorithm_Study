# 풀이 방법 요약
## 백 트래킹 사용 (14502.py)

- 백 트래킹으로 재귀호출 하면서 벽을 3개 세운다
- 벽 3개 세웠으면 msBFS로 바이러스 퍼트리기
- 안전지대 개수 세서 최댓값 갱신
---
## 조합 사용 (14502_2.py)

- 먼저 0인 칸의 좌표를 한 리스트에 저장, 그 리스트에서 3개를 뽑는 조합 사용
- 벽을 세우고 나면 백트래킹과 동일
