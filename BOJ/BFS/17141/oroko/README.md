## 💡 풀이 방법 요약

> 조합으로 바이러스 놓고 bfs로 퍼뜨려서 모든 칸에 퍼뜨리는 최소 시간 구하기

1. 0과 2의 개수 저장하기 (zero) => 모든 칸에 퍼뜨려졌는지 확인하기 위함
2. 조합으로 바이러스 놓을 곳 정하기
3. bfs로 퍼뜨리고 (zero-M)개의 칸에 퍼뜨렸으면 최소 시간 반환, 아니면 -1 반환

### perm
* 조합으로 바이러스 놓을 곳 정하기
* 바이러스 놓을 곳 M개를 큐에 저장하고 visited에 방문표시 해서 bfs에 넘겨준다.

### bfs
* myzero = zero - M 즉, 바이러스를 놓을 수 있는 곳 중에서 이미 놓여있는 곳을 뺀 수
* 방문할 때마다 myzero를 감소시킨다.
* myzero == 0 즉 모든 곳에 놓았으면 max, 아니면 -1 반환
