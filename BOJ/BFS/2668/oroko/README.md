## 💡 풀이 방법 요약
> 각 수에 대해 dfs로 순환이 있는지 확인해서 순환이면 선택하기

1. 1~N에 대하여 뽑히지 않았으면 dfs
2. dfs
   1. 해당 정점이 시작점으로 돌아온 점이면 오는 길에 방문한 노드들 선택하고 끝
   2. 방문표시 하고 다음 정점이 방문하지 않았거나 시작점이면 탐색
3. 선택한 수들을 정렬해서 출력
