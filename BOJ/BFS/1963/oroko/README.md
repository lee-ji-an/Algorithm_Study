## 💡 풀이 방법 요약

> bfs로 A에서 B가 될 때까지 탐색하기

### isPrime
* 소수인지 판별

### intToArray
* 정수를 배열 형태로 변환
* 예) 1000 -> { 1, 0, 0, 0 }
* 각 자리수를 비교하고 바꿀 때 사용

### arrayToInt
* 배열 형태의 수를 정수로 변환
* 예) { 1, 0, 0, 0 } -> 1000
* 정수로 저장하고 관리

### bfs
* 방문한 지 여부는 Set\<Integer\>으로 관리
* 각 네 자리에 대하여 0~9를 넣어서 소수이고 탐색한 지 없으면 탐색
