`곱하는 수가 고르게 분배되어있을 때 가장 크다(?)`

> N에 대해 곱하기를 2개로 갈랐을 때, 3개로 갈랐을 때 , .... 계속해보고 값이 줄어드는 순간의 그 이전 값이 최대  

`곱하기 수 정하는 방법` :   
t 개로 나눴을 때 곱하는 수가 (N - t + 1) 이 되어야 함 (계산해본 결과 그렇습니다)  
(N - t + 1) 을 t 개로 분할해야하니 (N - t + 1) 을 t 로 나눔  
몫 : s / 나머지 : r => (s + 1 -> r 개) / (s -> t - r 개)
