## Info
[12865 평범한 배낭](https://www.acmicpc.net/problem/12865)

## 💡 풀이 방법 요약
> dp[ i ][ j ] : i 번째 물건까지 고려하고 j 무게까지 담을 수 있을 때 물건 가치의 최댓값

현재 고려해야하는 물건의 무게가 w 이고 가치가 v일 때

점화식 : 
- ```dp[i][j] = dp[i-1][j], dp[i-1][j-w] + v```
- i 번째 물건을 넣지 않는것 / i 번째 물건을 넣는 것 둘 중 큰 값을 저장

## 🙂 마무리
1. 이 물건을 넣었을 때
2. 이 물건을 넣지 않았을 때
