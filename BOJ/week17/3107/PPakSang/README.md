## Info
[3107 - IPv6](https://www.acmicpc.net/problem/3107)

## 💡 풀이 방법 요약

':' 를 기준으로 문자열을 자르고, 공백 문자열이 아닌 문자열은 (4 - 문자열길이) 만큼 0 을 붙이고
(8 - 공백 문자열이 아닌 문자열 갯수) 만큼 '::' 로 이어진 영역을 '0000' 으로 채운다

## 🙂 마무리
자바 split
1. ::1.split(":") -> ["", "", "1"]
2. 1::.split(":") -> ["1"] 

이라서 후위 :: 에 대해서 갯수를 따로 구했다 -> last
