## 💡 풀이 방법 요약
> 한 라인 표현 
> 
>(x=A, B<y<C) or (y = A, B<x<C) -> 이것이 총 8개 필요

> 
> 행렬 회전
> 
> 방향 주고 라인 위에 있을 때 진행, 없으면 방향 꺾어서 진행 -> next 도출
> 
> 이 때 temp = next, next = prev, prev = next 사용
