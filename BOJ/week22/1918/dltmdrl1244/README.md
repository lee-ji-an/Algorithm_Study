## 💡 풀이 방법 요약
> 스택과 디큐를 멋지게 사용
- 사칙연산의 우선순위는 덧셈뺄셈 < 곱셈나눗셈 < 괄호
- 괄호 안을 먼저 처리. 스택에다가 하나씩 넣어가다가 닫는 괄호를 만나면 `make_substring` 함수를 이용해서 괄호 안을 postfix 로 만든다
- 괄호를 모두 제거하면 전체 operand에 대해 `make_substring` 을 한번 수행해서 합쳐준다.
